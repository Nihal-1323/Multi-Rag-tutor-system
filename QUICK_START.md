# ⚡ Quick Start - 3 Steps

## 1️⃣ Install Dependencies (One-Time)

```bash
# Backend
cd backend
pip install -r requirements.txt
cd ..

# Frontend
npm install
```

## 2️⃣ Start Application

**Terminal 1 - Backend:**
```bash
cd backend
python main.py
```
✅ Wait for: `INFO: Application startup complete`

**Terminal 2 - Frontend:**
```bash
npm run dev
```
✅ Wait for: `Local: http://localhost:3000/`

## 3️⃣ Open Browser

Go to: **http://localhost:3000**

---

## 🎯 That's It!

Now you can:
- ✅ Upload PDF files
- ✅ Ask questions
- ✅ See knowledge graph
- ✅ Get answers from your documents

---

## 🛑 To Stop

Press `Ctrl + C` in both terminals

---

## 🧪 Run Tests

```bash
# Backend tests
cd backend
pytest -v

# Frontend tests
npm test
```
